# SEO Seed Plan: Title, Meta Description & Focus Keyword

## Goal
Seed Yoast SEO fields (`_yoast_wpseo_title`, `_yoast_wpseo_metadesc`, `_yoast_wpseo_focuskw`) for all helmets (1,733), brands (99), and accessories (1,191) to improve search visibility and indexing. Compatible with Yoast SEO; follows best practices for length and keyword placement.

## Best Practices (Applied)
- **SEO Title**: 50–60 characters; primary keyword near the start; brand + product + type.
- **Meta Description**: 150–160 characters; include primary keyword, value prop or CTA; no duplicate titles.
- **Focus Keyword**: One main phrase per page (e.g. "Shoei RF-1400 helmet"); used by Yoast for analysis only (not output in page source).
- **Uniqueness**: Every title and description is unique per URL to avoid duplicate content signals.
- **Readability**: Natural language; avoid keyword stuffing.

## 1. Helmets

### Data used per helmet
- Post title (model name, e.g. "Shoei RF-1400 Matte Black")
- Brand name (from `rel_brand`)
- Helmet type (taxonomy `helmet_type`, e.g. Full Face, Modular)
- Certifications (taxonomy `certification`, e.g. ECE 22.06, DOT)
- Price (`price_retail_usd`)
- Helmet family (`helmet_family`, e.g. RF-1400)

### SEO Title (target ≤60 chars)
- **Pattern**: `{Brand} {Model} | {Type} Helmet | Helmetsan`
- **Fallback**: `{Post title} | Motorcycle Helmet | Helmetsan`
- Truncate model if needed to stay under 60 chars; prefer keeping brand + type.

### Meta Description (target 150–160 chars)
- **Pattern**: `Compare the {Brand} {Model} {type} – specs, safety ratings & sizes. {Certifications}. From ${price}. Find the best deal at Helmetsan.`
- Benefit-led, keyword-rich; certifications and price when available; single truncate at 160 chars.

### Focus Keyword
- **Primary**: `{Brand} {Model}` (e.g. "Shoei RF-1400")
- **Fallback**: `{Brand} helmet` or post title stem.

## 2. Brands

### Data used per brand
- Post title (brand name)
- Origin country (`brand_origin_country`)
- Helmet types offered (`brand_helmet_types` or taxonomy)
- Count of helmets (optional, for “X models”)

### SEO Title (target ≤60 chars)
- **Pattern**: `{Brand} Helmets | Official Hub & Reviews | Helmetsan`
- **Alternative**: `{Brand} Motorcycle Helmets | Prices & Reviews | Helmetsan`

### Meta Description (target 150–160 chars)
- **Pattern**: `Your hub for {Brand} ({Country}) helmets: full face, modular & more. Compare prices, certifications & where to buy. Official reviews at Helmetsan.`

### Focus Keyword
- **Primary**: `{Brand} helmets` (e.g. "Shoei helmets")
- **Alternative**: `{Brand} motorcycle helmets`

---

## 3. Accessories

### Data used per accessory
- Post title (product name)
- Type (`accessory_type`)
- Parent category / subcategory (`accessory_parent_category`, `accessory_subcategory` or taxonomy `accessory_category`)

### SEO Title (target ≤60 chars)
- **Pattern**: `{Title} | {Category} | Helmetsan`
- **Fallback**: `{Title} | Motorcycle Accessory | Helmetsan`
- Category from parent or subcategory or type.

### Meta Description (target 150–160 chars)
- **Pattern**: `Shop {title} – {category} for motorcycle helmets. Compatibility, reviews & buying guide at Helmetsan.`

### Focus Keyword
- **Primary**: `{Product name}` or `{category} {product}` (e.g. "Pinlock insert", "Bluetooth headset")

## Implementation

- **Service**: `Helmetsan\Core\Seo\YoastSeoSeeder` generates title, metadesc, focuskw per entity and updates Yoast post meta.
- **CLI**: `wp helmetsan seo seed [--post-type=helmet|brand|accessory|all] [--batch-size=N] [--limit=N] [--dry-run] [--use-ai]`
- **Yoast compatibility**: Only writes `_yoast_wpseo_title`, `_yoast_wpseo_metadesc`, `_yoast_wpseo_focuskw`. Yoast will use these; indexables can update on next crawl or when post is saved.
- **Idempotent**: Re-running seed overwrites with the same logic; safe to re-run after data changes.

## AI-powered meta descriptions (optional)

With `--use-ai`, meta descriptions are generated by **Groq** and **Google Gemini** (free tiers only). No cost when staying within limits.

### Cost: free tier only

- **Groq** (llama-3.1-8b-instant): ~14,400 requests/day free; no credit card.
- **Gemini** (gemini-1.5-flash): ~250 requests/day free; no credit card.
- Load balancing: even `post_id` → Groq first; odd → Gemini first. If one fails (e.g. rate limit), the other is tried. Full run (~3k items) uses both; after Gemini’s ~250/day, fallback goes to Groq. **You incur no charge** as long as you stay within these free limits.

### Setup

If you ever shared API keys in chat or in the repo, rotate them in Groq Console and Google AI Studio. In `wp-config.php` (do not commit real keys; use env or secrets in production):

```php
define('HELMETSAN_GROQ_API_KEY', 'your-groq-key');
define('HELMETSAN_GEMINI_API_KEY', 'your-gemini-key');
```

Or set env vars: `GROQ_API_KEY`, `GEMINI_API_KEY`.

### Running on server (production)

WordPress root on production: `/var/www/helmetsan.com/public/`. Run from that directory.

**Template-only (no AI):**
```bash
cd /var/www/helmetsan.com/public
wp helmetsan seo seed
```

**With AI (Groq + Gemini).** Set keys in `wp-config.php` or pass env vars:
```bash
cd /var/www/helmetsan.com/public
HELMETSAN_GROQ_API_KEY='your-key' HELMETSAN_GEMINI_API_KEY='your-key' wp helmetsan seo seed --use-ai
```

**Test first (5 items):**
```bash
wp helmetsan seo seed --use-ai --limit=5
```

### Faster runs (same PHP, no extra stack)

- **Parallelize manually**: Run two (or more) terminals with different ranges. Example: Terminal 1: `wp helmetsan seo seed --use-ai --post-type=helmet --offset=0 --limit=900`; Terminal 2: `wp helmetsan seo seed --use-ai --post-type=helmet --offset=900 --limit=900`. Each process uses its own rate-limit delay; total wall-clock time is roughly halved. Stay under 14.4k Groq + 250 Gemini per day across all processes.
- **Why PHP**: The command runs inside WordPress (WP-CLI); no extra runtime. For more concurrency you’d need a separate script (e.g. Node/Python) and a way to write Yoast meta (e.g. WP-CLI or REST). The current approach keeps everything in one stack and is sufficient for free-tier throughput.
